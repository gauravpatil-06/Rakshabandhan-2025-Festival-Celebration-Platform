## 🌟 Introduction

**Rakshabandhan 2025** is a modern festival celebration platform designed to bring siblings closer, no matter the distance. Instead of just a simple website, it's a digital ecosystem where you can explore gifts, share heartfelt wishes, and store precious memories in a beautiful, glassmorphic interface.

This project focuses on providing a professional-grade user experience with seamless navigation, real-time interactivity, and a robust backend integration.

---

## 🚀 How It Works: The Flow

1.  **Festive Entry**: Start your journey on a beautifully designed home page that captures the essence of Rakshabandhan.
2.  **Gift Exploration**: Browse through a curated collection of Rakhis and gifts with detailed descriptions.
3.  **Personalized Wishes**: Write and send personalized greetings to your siblings through the integrated "Wishes Wall."
4.  **Memory Vault**: Upload and preserve your celebration photos in the digital gallery.
5.  **Smart Wishlist**: Save your favorite items for later and manage your selections easily.
6.  **Order Tracking**: Keep track of your gift selections and order history in one place.

## 🛠️ Technologies Used

> **Tools and technologies used in this project**

*   **Frontend**: Next.js 14 (App Router) + React for a scalable architecture.
*   **Styling**: Tailwind CSS + Shadcn UI (Radix UI) for professional design tokens.
*   **Backend**: Supabase (PostgreSQL) for secure data management.
*   **Animations**: Framer Motion for premium micro-interactions and transitions.
*   **Icons**: Lucide React for consistent and modern iconography.
*   **State Management**: React Hooks & Context API for efficient local state.

## ⚙️ Installation & Setup

### 1. Repository Setup
```bash
git clone https://github.com/gauravpatil-06/Rakshabandhan-2025-Festival-Celebration-Platform.git
cd Rakshabandhan-2025-Festival-Celebration-Platform
```

### 2. Dependency Management
```bash
# Install dependencies
npm install
```

### 3. Environment Configuration
Create a `.env.local` file in the root directory:
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 4. Launching the App
```bash
# Start development server
npm run dev
```
